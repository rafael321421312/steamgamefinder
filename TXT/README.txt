🎮 Steam Game Finder
Steam Game Finder - это десктопное приложение с графическим интерфейсом (GUI), предназначенное для поиска игр в магазине Steam, отслеживания актуальных цен, просмотра системных требований и ведения списка избранных игр.

✨ Основной функционал
* Поиск игр: Поиск по базе Steam с использованием официальных API и веб-скрапинга.
* Актуальные цены: Отображение реальных цен в рублях (RUB) с учетом текущих скидок.
* Фильтрация и сортировка: Сортировка по цене, размеру скидки или релевантности. Фильтрация по жанрам.
* Детальная информация: Просмотр даты релиза, жанров, наличия демо-версии.
* Системные требования: Просмотр минимальных и рекомендуемых требований к ПК.
* История цен: Отображение прогнозируемых периодов лучших скидок (Летняя, Зимняя распродажи и т.д.).
* Избранное: Сохранение интересных игр в локальный список с автоматическим обновлением их цен.
* Темизация: 6 встроенных тем оформления (Тёмная, Светлая, Киберпанк, Океан, Закат, Лес).
* Быстрые действия: Открытие страницы игры в Steam или SteamDB в один клик.

📖 Руководство пользователя
1. Запуск и интерфейс
После запуска приложение открывается в центре экрана. Интерфейс разделен на три основные зоны:
*Верхняя панель: Поисковая строка, кнопки фильтров, сортировки и выбора темы.
*Левая панель: Список найденных игр с указанием их текущей цены.
* Правая панель: Вкладки с детальной информацией о выбранной игре (Информация, История цен, Характеристики, Избранное).

2. Как искать игры
1. Введите название игры (на русском или английском) в строку "Название игры"
2. Нажмите кнопку *🔍 Искать* или клавишу `Enter` на клавиатуре.
3. Дождитесь появления списка в левой панели.

3. Работа с результатами
* Клик по игре: Выберите игру в списке левой панели. В правой части загрузится информация о ней.
* Двойной клик Двойной клик по названию игры автоматически откроет её страницу в браузере.
* Кнопки действий:
    *🌐 Открыть в Steam` - открывает магазин Steam.
    *📊 SteamDB` - открывает страницу игры на сайте SteamDB (статистика, история цен).
    *⭐ В избранное` - добавляет игру в ваш личный список.

4. Фильтры и сортировка
*Сортировка: Выберите переключатель (Релевантность, Цена ↑, Цена ↓, Скидка).
*Жанр: Выберите жанр из выпадающего списка, чтобы отфильтровать результаты.
*Сброс: Нажмите `🔄 Сбросить фильтры`, чтобы вернуть исходный вид.

5. Горячие клавиши
*`Ctrl + F` - Переместить фокус в строку поиска.
*`F5` - Обновить цену для текущей выбранной игры.
*`Enter` - Начать поиск.
